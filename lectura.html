Inservible

EL objetivo del OVI es brindar conocimientos iniciales acerca del diseño de sitios Web en la actualidad a los estudiantes de la Universidad o a quien quiera que consulte o les llegue este material, basandonos en lenguaje HTML5 y CSS3 de una manera practica y sencilla.

Consejos practicos para aprender HTML

Disponga del tiempo necesario y no se apresure a entenderlo todo de una vez.

No use editores automaticos de codigo, no es bueno para aprender , ya cuando tenga mas experencia si es bueno que los use para facilitar el trabajo.

Maneje siempre el estandar de desarrollo, esto permitira realizar modificaciones mas faciles a la hora de actualiar su diseño.
