Mas de HTML5 y a practicar!

En la red se encuentran miles de paginas que nos ayudar a aprender a desarrollar sitios Web con HTML5 y CSS3, pero en la busqueda del mismo a continuacion les referencio una de las mejores webs que yo encontre donde tambien puedes practicar lo que vas aprendiendo. Lo genial de esta pagina es que la página te muestra en tiempo real lo que vas codificando en HTML y como se vería en un sitio web al instante.

Consejos practicos para aprender HTML

Disponga del tiempo necesario y no se apresure a entenderlo todo de una vez.

No use editores automaticos de codigo, no es bueno para aprender , ya cuando tenga mas experencia si es bueno que los use para facilitar el trabajo.

Maneje siempre el estandar de desarrollo, esto permitira realizar modificaciones mas faciles a la hora de actualiar su diseño.
